# AI Supported Secure Login System

## Project Description
This project is a secure login system enhanced with basic AI-based anomaly detection.

## Features
- User authentication system (Flutter)
- Backend API with Flask
- Brute-force attack simulation
- Weak password detection
- Suspicious login detection (AI logic)

##  Technologies
- Flutter
- Python (Flask)
- Basic Machine Learning Logic

##  Purpose
To demonstrate security vulnerabilities and how AI can be used to detect suspicious login attempts.
